# Notebooks

Use this folder for exploration notebooks only. Keep the main pipeline in `src/` and `scripts/` so the project remains runnable from the command line.

Suggested notebooks:

1. `01_signal_exploration.ipynb`: visualize raw EEG windows and filtered signals.
2. `02_feature_analysis.ipynb`: compare bandpower distributions across labels.
3. `03_model_error_analysis.ipynb`: inspect confusion matrix errors and hard examples.
